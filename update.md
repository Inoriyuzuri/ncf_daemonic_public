# NCF 更新指南

## 更新方式

### 方式一：Git 拉取（推荐）

如果你是通过 `git clone` 安装的：

```bash
cd 你的NCF目录
git pull
```

拉取完成后重新进入游戏或 `/reload` 即可。

### 方式二：手动替换

1. 下载最新版本的文件
2. 用新的 `ncf_encrypted.lua` 替换旧的
3. 用新的 `rotations/` 文件夹替换旧的
4. 重新进入游戏或 `/reload`

> **注意：** 不要删除 `config/` 文件夹，里面保存了你的授权码和个人设置。

## 更新后注意事项

- 配置文件 (`config/ncf_config.dat`) 会自动兼容，无需手动修改
- 授权码 (`config/ncf_license.dat`) 不受影响，无需重新输入
- 如果更新后出现异常，尝试 `/reload` 重载界面
- 如果仍有问题，可以删除 `config/ncf_config.dat`（不要删 license），重新进入游戏恢复默认设置

## 查看当前版本

进入游戏后，NCF 面板会显示当前版本号。
